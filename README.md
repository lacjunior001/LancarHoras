# LancarHoras
Projeto Pessoal...
A ideia é que este app seja capaz de facilitar a vida em alguns dos procedimentos internos da empresa que trabalho.
através dele a tarefa de lançar horas trabalhadas em projetos, será feita de forma mais rápida e prática.

Será feita conexão via API com o sistema de gestão de projetos da empresa, e o app será capaz de listar os projetos em que o usuário está alocado, e permitir que o mesmo lance as horas trabalhadas em cada projeto.

Ainda em fase bem inicial pois é meu primeiro contato com .NET Maui.
